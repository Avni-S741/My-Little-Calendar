🌸 Avnii's Calendar – Electron App

A cute and minimal desktop calendar app built using the Electron framework.
This app displays the current day and month in a soft pastel UI design.

I created this project as part of my learning journey with Electron, inspired by the amazing creator Nasha Wanich who has very creative UI style.....

✨ Features

Displays current date (Day + Month)

Soft pastel, kawaii-inspired UI

Lightweight desktop window

Draggable frameless window

Built entirely using:

HTML

CSS

JavaScript

Electron
